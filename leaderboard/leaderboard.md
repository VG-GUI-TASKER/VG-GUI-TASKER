## Metric Definitions

- **Acc.**: Overall accuracy = Σ 𝟙(type_match) × (0.3 + 0.7 × 𝟙(param_match)) / N
- **Type Acc.**: Proportion of steps where predicted action type matches ground truth
- **Comp.**: Episode Completion = avg(correct_steps / total_steps) per episode
- **Eff.↓**: Average number of input frames per prediction step (lower = more efficient)
- **PIR**: Performance Improvement Rate = (Acc_video − Acc_no_video) / Acc_no_video
